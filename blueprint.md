# 수학 문제집 판매 웹사이트 프로젝트

## 1. 개요

이 프로젝트는 사용자가 수학 문제집을 둘러보고 구매할 수 있는 간단하고 현대적인 전자 상거래 웹사이트를 구축하는 것을 목표로 합니다. 최신 웹 표준(HTML, CSS, JavaScript)과 웹 컴포넌트를 사용하여 프레임워크 없이 제작됩니다.

## 2. 디자인 및 기능

### 디자인
- **색상 팔레트:** 교육적인 신뢰감을 주는 파란색과 녹색 계열을 주 색상으로 사용하고, 활기찬 노란색을 포인트 색상으로 활용하여 학습의 즐거움을 표현합니다.
- **타이포그래피:** 가독성이 좋은 'Noto Sans KR' 폰트를 기본으로 사용합니다. 제목과 본문에 글꼴 크기와 굵기를 조절하여 위계질서를 명확히 합니다.
- **레이아웃:** CSS 그리드와 Flexbox를 사용하여 반응형 레이아웃을 구현합니다. 제품 카드는 그리드 형태로 정렬하여 사용자가 쉽게 훑어볼 수 있도록 합니다.
- **시각 효과:** 카드에 부드러운 그림자 효과를 주어 입체감을 더하고, 버튼에 호버 효과를 추가하여 상호작용성을 높입니다.

### 기능
- **웹 컴포넌트 기반 제품 카드:** 각 문제집은 `product-card`라는 맞춤형 웹 컴포넌트로 표시됩니다. 이 컴포넌트는 이미지, 제목, 가격, 장바구니 담기 버튼을 포함합니다.
- **반응형 디자인:** 모바일, 태블릿, 데스크톱 등 다양한 화면 크기에서 최적의 사용자 경험을 제공합니다.
- **동적 콘텐츠:** JavaScript를 사용하여 제품 데이터를 동적으로 페이지에 렌더링합니다.

## 3. 현재 구현 계획

1. **`index.html` 구조 설정:**
    - 기본 HTML5 구조를 설정합니다.
    - 한국어 사용을 위해 `lang="ko"` 속성을 추가합니다.
    - `style.css`와 `main.js` 파일을 연결합니다.
    - 헤더, 메인, 푸터 영역으로 구조를 나눕니다.
2. **`style.css` 스타일링:**
    - 전반적인 디자인 시스템(색상, 폰트)을 CSS 변수로 정의합니다.
    - 각 섹션(헤더, 메인, 푸터)과 제품 카드 레이아웃을 스타일링합니다.
    - 미디어 쿼리를 사용하여 반응형 디자인을 구현합니다.
3. **`main.js` 기능 구현:**
    - `product-card` 웹 컴포넌트를 정의합니다.
    - 샘플 제품 데이터를 생성합니다.
    - 제품 데이터를 기반으로 `product-card` 인스턴스를 동적으로 생성하여 페이지에 추가합니다.
